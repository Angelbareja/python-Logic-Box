# Pattern Generator and Number Analyzer

## Overview
A beginner-friendly Python console application that demonstrates the use of loops, conditional statements, user input handling, and basic arithmetic operations.

The project allows users to:
- Generate a star (`*`) pattern.
- Analyze numbers within a given range.
- Identify whether numbers are even or odd.
- Calculate the sum of all numbers in a selected range.

## Features

### 1. Pattern Generator
- Accepts the number of rows from the user.
- Displays a right-angled triangle pattern using asterisks (`*`).

Example:
```
*
**
***
****
*****
```

### 2. Number Analyzer
- Takes a start and end value from the user.
- Determines whether each number is even or odd.
- Calculates the total sum of all numbers in the range.

Example:
```
Number 1 is Odd
Number 2 is Even
Number 3 is Odd
...
Sum of all numbers from 1 to 5 is: 15
```

### 3. Menu-Driven Interface
- Easy-to-use interactive menu.
- Continuous execution until the user chooses to exit.
